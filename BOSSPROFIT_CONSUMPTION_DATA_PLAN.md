# BOSSPROFIT 소비량 데이터 연결안

## 결론

농수산물의 “오늘 소비량”을 직접 제공하는 단일 공개 API는 확인하기 어렵다. BOSSPROFIT은 서로 의미와 갱신 주기가 다른 세 종류를 분리한다.

| 구분 | 후보 출처 | 주기 | 화면 용도 |
|---|---|---|---|
| 공식 소비 수준 | KOSIS 식품수급 관련 통계 | 연간 | 1인당 공급량·장기 소비 추세 |
| 실제 구매 행동 | 농촌진흥청 농식품 소비자패널 | 월·연 | 가구 구매량·구매빈도·계절 소비 |
| 일별 수요 대리 지표 | 농림축산식품부·aT 공영도매시장 거래량 | 실시간·일별 | 거래량 TOP3와 단기 수급 강도 |

## 사용 원칙

- 도매시장 거래량을 소비량이라고 부르지 않는다.
- 공식 소비 수준은 연간 통계의 기준연도와 단위를 표시한다.
- 소비자패널은 표본 기반 자료임을 표시한다.
- 품목·품종·등급·단위가 다르면 같은 순위에서 직접 비교하지 않는다.
- 실제 API 연결 전 UI 값은 `--` 또는 `연동 후 표시`로 표현한다.

## 필요한 인증

```env
KOSIS_API_KEY=
DATA_GO_KR_SERVICE_KEY=
```

농식품 소비자패널은 공개 범위와 원자료 이용 절차를 별도로 확인한다.

## 화면

`/market`

- 시장 브리핑
- 소비량
- 데이터 출처
- 거래량 TOP3
- 오늘 가격 변동 TOP3
- 내일 예상 가격 변동 TOP3
